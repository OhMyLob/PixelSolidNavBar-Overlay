# PixelSolidNavBar-Overlay

Note: this is tested only on a Nexus 5X

Enable Pixel solid NavBar icons.
Download this repo, open a Terminal/Command Prompt and execute these commands:

Linux/MacOS:
- cd PATH_TO_DOWNLOADED_REPO
- cd scripts
- sh install.sh

Windows:
- cd PATH_TO_DOWNLOADED_REPO
- cd scripts
- bat install.bat
